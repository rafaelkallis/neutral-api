<h1 align="center">Covee Network 2.0</h1>

## Usage

```bash

# prepare app
$ npm install
$ docker-compose up

# use for development (also watches for changes!)
$ npm run dev

# use for production
$ npm run build
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## OpenApi Spec

Start the app and visit `/docs` if you are a human , or `/docs-json` if you are a machine.
