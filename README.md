<h1 align="center">Covee Network 2.0</h1>

## Usage

```bash

$ npm install
$ docker-compose up

# development
$ npm run dev

# development watch mode
$ npm run dev:watch

# production mode
$ npm run build
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
